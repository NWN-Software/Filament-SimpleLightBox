Does not rely on any third-party JavaScript libraries.

## Installation

You can install the package via composer:

```bash
composer require solutionforest/filament-simplelightbox
```

## Usage

```php
Tables\Columns\ImageColumn::make('image')
                    ->simpleLightbox()
```
